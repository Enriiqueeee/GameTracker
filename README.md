## Aplicación de Gestión de Videojuegos con RAWG API 🎮

Esta aplicación Android permite gestionar una biblioteca de videojuegos utilizando la API de RAWG. Los usuarios pueden buscar juegos, filtrarlos por género o nombre, marcar favoritos y acceder a recomendaciones de juegos similares. Además, ofrece almacenamiento local para mantener un registro de los juegos guardados.

## Características

📃 Listado de Videojuegos: Consulta una lista de juegos desde la API de RAWG.

🔍 Búsqueda y Filtrado: Encuentra juegos por nombre o género.

✨ Recomendaciones: Descubre juegos similares según tus preferencias.

⭐ Favoritos y Jugados: Guarda juegos en listas personalizadas.

💾 Almacenamiento Local: Integración con Room, SharedPreferences y Firebase.

🌐 Conectividad con API: Uso de Retrofit para consumir los datos.

🛠️ Arquitectura MVVM: Patrón para una mejor organización del código.
